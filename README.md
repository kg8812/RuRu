# RuRu

캐릭터를 타이밍에 맞춰 쌓아 올리는 미니 게임입니다.

<br/>

---

## 게임 소개

RuRu는 캐릭터를 정확한 타이밍에 맞춰 위로 쌓아 올리는 캐주얼 게임입니다.

플레이어는 적절한 순간에 입력을 하여 캐릭터를 쌓아야 하며,  
타이밍이 어긋나면 쌓기에 실패하며 게임이 종료됩니다.

게임이 진행될수록 속도가 점점 빨라지며,  
아이템을 획득해 새로운 능력이 발동되는 재미가 있습니다.

<br/>

---

## 주요 기능

- 타이밍 기반 캐릭터 배치 시스템
- 캐릭터 누적 스택 구조
- 진행에 따른 속도 증가
- 아이템 시스템 (플레이 보조 요소)
- 타이밍 실패 시 게임 종료 처리
- 점수 또는 높이 기반 진행 시스템
- 간단한 입력 기반 조작
- UI를 통한 게임 상태 표시

<br/>

---

## 플레이 목표

- 최대한 많은 캐릭터를 쌓기
- 점점 빨라지는 속도에 적응하며 높은 기록 달성
- 아이템을 활용해 더 높은 스택 유지
